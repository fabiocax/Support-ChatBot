# Support-ChatBot-Python


Instalação:

```
$ python3 -m venv Support-ChatBot
$ cd Support-ChatBot/
$ source bin/activate
$ git clone https://github.com/leonardogt4/Support-ChatBot.git
$ cd Support-ChatBot
$ pip3 install -r requirements.txt 

```
